4 Development Board Application
5 ========================
6 Part of the Smart Music Player project. Program for the development board.
7 [Github project](https://github.com/VAMK-embedded-project-2019A/Development-Board-Application).
8 
9 Requirements
10 ------------------------
11 * Raspberry Pi 3B+ running Raspbian Buster with desktop and recommended software (September 2019)
12 * Microstack GPS module.
13 * Deltaco USB sound card 7.1
14 
15 Install dependencies
16 ------------------------
17 ```console
18 [pi@raspberry:~]$ sudo apt install libssl-dev=1.1.1c-1
19 [pi@raspberry:~]$ sudo apt install libjsoncpp-dev=1.7.4-3
20 [pi@raspberry:~]$ sudo apt install libcurl4-openssl-dev=7.64.0-4
21 [pi@raspberry:~]$ sudo apt install libmpg123-dev=1.25.10-2
22 ```
