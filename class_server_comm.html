◆ start()

bool ServerComm::start ( )

Start communicating with the server.

If the weather tag has not been updated in the last 1 minute, call getGpsLocation() and then getWeatherTag(). Subsequently call getSongInfo() and choose a random song in the returned vector to call downloadSong().

Returns
True if none operation failed, false otherwise.
